== Custom Fields in Flow

If you have multiple components that you want to act as a single field, or just a component that does not have support for use with Binder, setting a label, setting a helper message, you will likely want to wrap those components within a Custom Field.
To create a Custom Field in Flow you will need to create a class extending `CustomField` and specify the value type, that will represent the combined value of all its child components.
You will also have to implement two methods `generateModelValue` (compute the Custom Field value from child components) and `setPresentationValue` (distribute the Custom Field value to child components). These methods will be called automatically once a value change is detected on client-side or server-side respectively.
