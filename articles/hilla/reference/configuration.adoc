== Core And Pro Vaadin React Components
Vaadin React components include free (core) components and commercial (pro) components, which are shipped in the separate npm packages: `@vaadin/react-components` (only free) and `@vaadin/react-components-pro` (only commercial).
Vaadin adds both of these packages to [file]`package.json`, if `com.vaadin:vaadin` artifact is in project's configuration:

[source,xml]
.pom.xml
----
<dependency>
	<groupId>com.vaadin</groupId>
	<artifactId>vaadin</artifactId>
</dependency>
----

and adds only `@vaadin/react-components`, if `com.vaadin:vaadin-core` is used:

[source,xml]
.pom.xml
----
<dependency>
	<groupId>com.vaadin</groupId>
	<artifactId>vaadin-core</artifactId>
</dependency>
----
