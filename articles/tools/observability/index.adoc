= Observability Kit

:commercial-feature: Observability Kit
:kit-start-trial: true
include::{articles}/_commercial-banner.adoc[opts=optional]

Observability Kit provides integration with industry-leading Application Performance Monitoring (APM) and observability tools to provide insight into both the frontend and backend of an application.

At its core, Observability Kit is built on https://micrometer.io[Micrometer].
It instruments the Vaadin runtime and records metrics into your application's [classname]`MeterRegistry`, while emitting tracing spans through the Micrometer Observation API -- so the data flows to whatever backend you already use.
With Spring Boot, it's a drop-in: add one dependency and you're done.

.Migrating From Version 4?
[NOTE]
Earlier versions of Observability Kit were a standalone OpenTelemetry Java agent.
See the <<migrating-to-v5#,Migrating to Version 5>> page if you're upgrading an existing project.
