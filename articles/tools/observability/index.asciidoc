= Observability Kit

:commercial-feature: Observability Kit
:kit-trial: true
include::{articles}/_commercial-banner.asciidoc[opts=optional]

Observability Kit provides integrations with industry-leading Application Performance Monitoring (APM) and observability tools.

At its core, Observability Kit is a custom Java agent that runs together with a Vaadin Flow application to automatically collect telemetry data, including traces, metrics, and logs.

Download the latest version from the following link:

https://repo1.maven.org/maven2/com/vaadin/observability-kit-agent/2.0.0/observability-kit-agent-2.0.0.jar[Download Observability Kit,role="button primary water"]
