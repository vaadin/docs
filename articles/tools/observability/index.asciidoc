= Observability Kit

:commercial-feature: Observability Kit
include::{articles}/_commercial-banner.asciidoc[opts=optional]

Observability Kit provides integrations with industry-leading Application Performance Monitoring (APM) and observability tools.

At its core, Observability Kit is a custom Java agent that runs together with a Vaadin Flow application to automatically collect telemetry data, including traces, metric, and logs.
