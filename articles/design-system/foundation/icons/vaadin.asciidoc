= Vaadin Icons

Vaadin Icons is a set of 600+ icons designed for web applications.

[source, typescript, role="render-only embedded"]
----
include::../../../../frontend/demo/foundation/vaadin-icons.ts[render, persistent]
----

The icons are drawn on a 16&times;16 pixel canvas.


== Usage

In server-side views (Flow/Java):

[source, java]
----
Icon icon = new Icon('vaadin:photo');
----

In client-side views (Fusion/TypeScript):

[source, typescript]
----
import { html, LitElement, customElement } from 'lit-element';
import '@vaadin/vaadin-icons/vaadin-icons';

@customElement('photo-icon-example')
export class PhotoIconExample extends LitElement {
  render() {
    return html`<iron-icon icon="vaadin:photo"></iron-icon>`;
  }
}
----

See the documentation for the <<../../components/icon#,Icon>> component for more information.


=== Advanced usage (Fusion/TypeScript usage)

If you need to be in control of when or how https://www.webcomponents.org/element/PolymerElements/iron-icon[iron-icon] is imported, you may import `iconset.js` instead of `icons.js`.
The `iconset.js` file imports only the icon definitions (https://www.webcomponents.org/element/PolymerElements/iron-iconset-svg[iron-iconset-svg]) without automatically importing the `iron-icon` element.
