= Lumo Colors

Lumo defines a set of style properties (CSS custom properties) for colors. You can use these properties to apply a consistent color palette across your application.

You can use the built-in <<{articles}/styling/lumo/lumo-variants#, Lumo light and dark mode variants>> to switch the color palette of all components in your application.

This page lists the available color properties in Lumo.


== Base

The base color is used as the main background color for the system. The <<grayscale>> colors are designed to be used on top of it, and the <<text>> colors are designed to provide sufficient contrast with it.

[.property-listing.previews, cols="2,>1"]
|===
| Description | CSS Custom Property

| [preview(--lumo-base-color)]*Base*
| [custom-property]#--lumo-base-color#

|===
