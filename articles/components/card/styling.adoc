=== Common Properties

[cols="1,2,2"]
|===
| Feature | Property | Default Value

|Background
|`--vaadin-card-background`
|`--lumo-shade-5pct`

|Box Shadow
|`--vaadin-card-box-shadow`
|`none`

|Border Width
|`--vaadin-card-border-width`
|`0`

|Border Color
|`--vaadin-card-border-color`
|`--lumo-contrast-20pct`

|Border Radius
|`--vaadin-card-border-radius`
|`--lumo-border-radius-l`

|Padding
|`--vaadin-card-padding`
|`--lumo-space-m`

|Gap
|`--vaadin-card-gap`
|`--lumo-space-m`

|Media Aspect Ratio
|`--vaadin-card-media-aspect-ratio`
|`16/9`, when the `stretch-media` or `cover-media` style variant is used without the `horizontal` style variant.

|===

=== Parts

Media:: `vaadin-card+++<wbr>+++**::part(media)**`
Header:: `vaadin-card+++<wbr>+++**::part(header)**`
Content:: `vaadin-card+++<wbr>+++**::part(content)**`

Title:: `vaadin-card +++<wbr>+++**[slot="title"]**`

=== Style Variants

Outlined:: `vaadin-card+++<wbr>+++**[theme~="outlined"]**`
Elevated:: `vaadin-card+++<wbr>+++**[theme~="elevated"]**`
Horizontal:: `vaadin-card+++<wbr>+++**[theme~="horizontal"]**`
Stretch Media:: `vaadin-card+++<wbr>+++**[theme~="stretch-media"]**`
Cover Media:: `vaadin-card+++<wbr>+++**[theme~="cover-media"]**`
