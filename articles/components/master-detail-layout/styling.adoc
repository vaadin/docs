Root element:: `vaadin-master-detail-layout`


=== States

Overlay mode:: `vaadin-master-detail-layout+++<wbr>+++**[overlay]**`
Stack mode:: `vaadin-master-detail-layout+++<wbr>+++**[stack]**`
Horizontal orientation:: `vaadin-master-detail-layout+++<wbr>+++**[orientation="horizontal"]**`
Vertical orientation:: `vaadin-master-detail-layout+++<wbr>+++**[orientation="vertical"]**`
Default (layout) containment:: `vaadin-master-detail-layout+++<wbr>+++**[containment="layout"]**`
Viewport containment:: `vaadin-master-detail-layout+++<wbr>+++**[containment="viewport"]**`
Layout with detail provided:: `vaadin-master-detail-layout+++<wbr>+++**[has-detail]**`


=== Parts

Master area:: `vaadin-master-detail-layout+++<wbr>+++**::part(master)**`
Detail area:: `vaadin-master-detail-layout+++<wbr>+++**::part(detail)**`
Backdrop in the overlay mode:: `vaadin-master-detail-layout+++<wbr>+++**::part(backdrop)**`
