Root element:: `vaadin-slider`


=== States

Focused:: `vaadin-slider+++<wbr>+++**[focused]**`
Keyboard focused:: `vaadin-slider+++<wbr>+++**[focus-ring]**`
Disabled:: `vaadin-slider+++<wbr>+++**[disabled]**`
Read-only:: `vaadin-slider+++<wbr>+++**[readonly]**`
Active:: `vaadin-slider+++<wbr>+++**[active]**`
Required:: `vaadin-slider+++<wbr>+++**[required]**`
Min/max labels visible:: `vaadin-slider+++<wbr>+++**[min-max-visible]**`


=== Parts

Track:: `vaadin-slider+++<wbr>+++**::part(track)**`
Track fill:: `vaadin-slider+++<wbr>+++**::part(track-fill)**`
Thumb:: `vaadin-slider+++<wbr>+++**::part(thumb)**`
Min/max label container:: `vaadin-slider+++<wbr>+++**::part(marks)**`
Min label:: `vaadin-slider+++<wbr>+++**::part(min)**`
Max label:: `vaadin-slider+++<wbr>+++**::part(max)**`
Label:: `vaadin-slider+++<wbr>+++**::part(label)**`
Helper text:: `vaadin-slider+++<wbr>+++**::part(helper-text)**`
Error message:: `vaadin-slider+++<wbr>+++**::part(error-message)**`
Required indicator:: `vaadin-slider+++<wbr>+++**::part(required-indicator)**`


=== Value Bubble

The value bubble is a separate `vaadin-slider-bubble` component with the following shadow DOM parts:

Overlay:: `vaadin-slider-bubble+++<wbr>+++**::part(overlay)**`
Content:: `vaadin-slider-bubble+++<wbr>+++**::part(content)**`
Arrow:: `vaadin-slider-bubble+++<wbr>+++**::part(arrow)**`
