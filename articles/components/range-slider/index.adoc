// tag::description[]
Range Slider is an input field that lets users select a numeric range by dragging two thumbs along a track. Its value is a pair of numbers representing the start and end of the selected range. Use arrow keys for precise adjustments.

== Best Practices

Range Slider is best suited for filtering or selecting a subset of a continuous range, such as a price range or date span. For precise numeric input, consider using two <<../number-field#,Number Fields>> instead.

When setting `min`, `max`, and `step`, choose values that result in a reasonable number of selectable steps. Too many steps make precise selection difficult; too few limit the user's options.
