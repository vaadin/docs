= Notification Styling

include::../_styling-section-intros.adoc[tag=selectors]


Root element:: `vaadin-notification-card`


=== Parts

Notification card:: `vaadin-notification-card+++<wbr>+++**::part(overlay)**`
Content wrapper:: `vaadin-notification-card+++<wbr>+++**::part(content)**`


=== Style Variants

Success:: `vaadin-notification-card+++<wbr>+++**[theme="success"]**`
Warning:: `vaadin-notification-card+++<wbr>+++**[theme="warning"]**`
Error:: `vaadin-notification-card+++<wbr>+++**[theme="error"]**`
Primary:: `vaadin-notification-card+++<wbr>+++**[theme="primary"]**`
Contrast:: `vaadin-notification-card+++<wbr>+++**[theme="contrast"]**`
