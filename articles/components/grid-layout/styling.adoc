= Grid Layout Styling

== Style Variants

Grid Layout supports the following style variants:

[cols="1,1"]
|===
| Variant | Supported by

|`margin`
|Aura, Lumo

|`padding`
|Aura, Lumo

|`spacing`
|Aura, Lumo

|`spacing-xs`
|Lumo

|`spacing-s`
|Lumo

|`spacing-l`
|Lumo

|`spacing-xl`
|Lumo

|===

== Spacing

Spacing creates gaps between items in the grid. You can enable spacing using theme variants or set custom gap values.

[.example]
--
ifdef::flow[]
[source,java]
----
include::{root}/src/main/java/com/vaadin/demo/component/gridlayout/GridLayoutSpacing.java[render,tags=snippet,indent=0,group=Flow]
----
endif::[]
--

Five different spacing style variants are available in Lumo theme:

|===
|Style Variant |Usage Recommendation

|`spacing-xs`
|Extra-small space between items.

|`spacing-s`
|Small space between items.

|`spacing`
|Medium space between items.

|`spacing-l`
|Large space between items.

|`spacing-xl`
|Extra-large space between items.
|===


[.example]
--
ifdef::flow[]
[source,java]
----
<source-info group="Flow"></source-info>
GridLayout layout = new GridLayout();
layout.addThemeVariants(GridLayoutVariant.LUMO_SPACING_XS);
----
endif::[]
--

Alternatively, set a custom gap value directly:

[.example]
--
ifdef::flow[]
[source,java]
----
<source-info group="Flow"></source-info>
GridLayout layout = new GridLayout();
layout.setGap("12px");

// Or set column and row gaps separately
layout.setColumnGap("20px");
layout.setRowGap("10px");
----
endif::[]
--
