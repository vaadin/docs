= Breadcrumbs Styling

== Style Variants

Breadcrumbs supports the following style variants.

[cols="1,3,1"]
|===
| Variant | Description | Supported by

|`slash`
|Uses a slash (`/`) as the separator between items instead of the default separator
|Lumo, Aura

|`primary`
|Renders the links using the primary color for stronger emphasis
|Lumo

|`accent`
|Renders the links using the accent color for stronger emphasis
|Aura

|===

Root element:: `vaadin-breadcrumbs`


== Parts

Item list:: `vaadin-breadcrumbs+++<wbr>+++**::part(list)**`
Overflow wrapper:: `vaadin-breadcrumbs+++<wbr>+++**::part(overflow)**`
Overflow button:: `vaadin-breadcrumbs+++<wbr>+++**::part(overflow-button)**`
Overflow overlay:: `vaadin-breadcrumbs+++<wbr>+++**::part(overlay)**`
Overflow overlay content:: `vaadin-breadcrumbs+++<wbr>+++**::part(overlay-content)**`

== States

With collapsed items:: `vaadin-breadcrumbs+++<wbr>+++**[has-overflow]**`


== Breadcrumbs Item

Item root element:: `vaadin-breadcrumbs-item`
Current page item:: `vaadin-breadcrumbs-item+++<wbr>+++**[current]**`
Item link (with path):: `vaadin-breadcrumbs-item+++<wbr>+++**::part(link)**`
Item non-link (current page):: `vaadin-breadcrumbs-item+++<wbr>+++**::part(nolink)**`
Item label:: `vaadin-breadcrumbs-item+++<wbr>+++**::part(label)**`
Item prefix element:: `vaadin-breadcrumbs-item+++<wbr>+++** > [slot="prefix"]**`
Item with prefix content:: `vaadin-breadcrumbs-item+++<wbr>+++**[has-prefix]**`
Disabled item:: `vaadin-breadcrumbs-item+++<wbr>+++**[disabled]**`
