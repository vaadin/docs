= Side Navigation

// tag::description[]
The Side Navigation component provides a vertical list of navigation links with support for collapsible, nested sections.
// end::description[]

.Preview Feature
[IMPORTANT]
====
This is a preview release of the Side Navigation component. Its behavior, API, and look and feel might still change. In order to use the component, it must be explicitly enabled with a feature flag. See the <<Feature Flag>> section for how to do this.
====

The Side Navigation component can be used for example in the drawer of an <<../app-layout#,App Layout>>.


== Prefix and Suffix Elements

Navigation items have slots for prefix and suffix elements. The prefix slot is primarily intended for icons, while the suffix slot can for example be used for notification badges.

Interactive prefix and suffix elements are not recommended, as the entire item row acts as a link.

== Hierarchy

Navigation items can contain sub-items, which are collapsed by default. There is no technical limitation on the number of nesting levels, but a maximum of three levels is recommended for usability reasons.

Parent items can be links, in which case clicking them expands their sub-items in addition to navigating. Non-link parent items can be achieved by omitting the target path.

== Scrolling

The Side Navigation component does not contain a scroll area. Instead, it can be made scrollable by wrapping it inside a <<../scroller#,Scroller>>.

== Feature Flag

In order to use the Side Navigation component, it must be explicitly enabled with a feature flag.
There are two methods of doing this:


=== Using Vaadin Developer Tools

1. Click on the Vaadin Developer Tools icon button in your running application
2. Open the [guilabel]`Experimental Features` tab
3. Enable the [guilabel]`SideNav component` feature
4. Restart the application.


=== Adding a Feature Flags Properties File

1. Create a [filename]#src/main/resources/vaadin-featureflags.properties# file in your application folder
2. Add the following content: `com.vaadin.experimental.sideNavComponent=true`
3. Restart the application.
