= Side Navigation

// tag::description[]
The Side Navigation component provides a vertical list of navigation links with support for collapsible, nested sections.
// end::description[]

.Preview Feature
[IMPORTANT]
====
This is a preview release of the Side Navigation component. Its behavior, API, and look and feel might change. In order to use this component, it must be explicitly enabled with a feature flag. See the <<Feature Flag>> section for how to do this.
====

== Prefix & Suffix Elements

Navigation items have slots for prefix and suffix elements. The prefix slot is primarily intended for icons, while the suffix slot can, for example, be used for notification badges.

Interactive prefix and suffix elements are not recommended since the entire item row acts as a link.

== Hierarchy

Navigation items can contain sub-items, which are collapsed by default. There's no technical limitation on the number of nesting levels, but a maximum of three levels is recommended for usability reasons.

Parent items can be links. Clicking them expands their sub-items in addition to navigating. Non-link parent items can be achieved by omitting the target path.

== Scrolling

The Side Navigation component does not contain a scroll area. Instead, it can be made scrollable by wrapping it inside a <<../scroller#,Scroller>>.


== Keyboard Usage

[cols="1,2"]
|===
|Shortcut |Function

|kbd:[Tab]
|Navigation between list items.

|kbd:[Tab]
|Navigation between link and expand/collapse button.

|kbd:[Enter] / kbd:[Space]
|Toggles expand/collapse.

|kbd:[Enter]
|Trigger link.

|===

== Feature Flag

In order to use the Side Navigation component, it must be explicitly enabled with a feature flag. To do this, there are two methods: using Vaadin Developer Tools; or by adding a Feature Flags properties file.

To use Developer Tools, click its icon button in your running application. Then open the [guilabel]`Experimental Features` tab. Enable the [guilabel]`SideNav component` feature and restart the application.

To add a Feature Flags properties file, first create a [filename]#src/main/resources/vaadin-featureflags.properties# file in your application folder. Add to it `com.vaadin.experimental.sideNavComponent=true`. Then restart the application.
