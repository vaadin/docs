= Progress Bar Styling

== Style Variants

Progress Bar comes with three style variants: contrast, success, and error.

[cols="1,3,1"]
|===
|Variant | Description | Supported by

| `success`
| When progress is satisfactory and/or nearing completion

| Lumo

| `error`
| When progress is unsatisfactory or draw the user's attention to a stalled or failed process
| Lumo

| `contrast`
| Improved visibility
| Lumo

|===
