= Form Layout Styling

include::../_styling-section-intros.adoc[tag=properties]

[cols="2,3,2"]
|===
| Feature | Property | Default Value

|Spacing between columns
|`--vaadin-form-layout-column-spacing`
|`--lumo-space-l`

|Spacing between rows
|`--vaadin-form-layout-row-spacing`
|`0`

|Spacing between side label and field
|`--vaadin-form-layout-label-spacing`
|`1em`

|Side label width
|`--vaadin-form-layout-label-width`
|`8em`
|===

include::../_styling-section-intros.adoc[tag=selectors]

Form Layout:: `vaadin-form-layout`
Form Row:: `vaadin-form-row`
Form Item:: `vaadin-form-item`
Form Item Label:: `vaadin-form-item+++<wbr>+++**::part(label)**`
Form Item Label Text:: `vaadin-form-item+++<wbr>+++** > label**`
Form Item Required Indicator:: `vaadin-form-item+++<wbr>+++**::part(required-indicator)**`
Form Item with Required Field:: `vaadin-form-item+++<wbr>+++**[required]**`
