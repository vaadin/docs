= Password Field

The input is masked by default. On mobile devices, the last typed letter is shown for a brief moment. The masking can be toggled using an optional reveal button.

[source,tsx]
----
<PasswordField label="Password" value="Ex@mplePassw0rd" />
----

== Reveal Button

----

  label="Password"
  value="Ex@mplePassw0rd"
