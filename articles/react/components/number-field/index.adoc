== Stepper Controls

Stepper controls allow the user to make small adjustments, quickly.

[source,tsx]
----
<HorizontalLayout theme="spacing">
  <NumberField label="Adults" stepButtonsVisible value="1" min={1} />
  <NumberField
    label="Children"
    stepButtonsVisible
    value="0"
    min={0}
    max={9}
    helperText="Age 2-12"
  />
  <NumberField label="Infants" stepButtonsVisible value="0" min={0} max={9} />
</HorizontalLayout>
----

== Step

The step value of a Number Field defines its numeric intervals. It specifies the amount by which the value increases or decreases when using the Up and Down arrow keys, or the stepper controls. It invalidates the field if the value entered doesn't align with the specified step.

[source,tsx]
----
<NumberField label="Duration (hours)" stepButtonsVisible step={0.5} value="12.5" />
----
