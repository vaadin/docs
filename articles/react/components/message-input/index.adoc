= Message Input

[source,tsx]
----
<MessageInput
  onSubmit={({ detail: { value } }) =>
    Notification.show(`Message received: ${value}`, { position: "middle" })
  }
/>
----

The user can send the message either by pressing kbd:[Enter] or by clicking the "Send" button -- use kbd:[Shift+Enter] to add a new line.

Use the <<../message-list#,Message List>> component to show messages that users have sent.

[source,tsx]
----
<MessageList items={items} />
<MessageInput
  onSubmit={({ detail: { value } }) =>
    setItems([
      ...items,
      {
        text: value,
        time: "seconds ago",
        userName: "Milla Sting",
        userColorIndex: 3,
      },
    ])
  }
/>
----
