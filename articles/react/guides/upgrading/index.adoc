== Frontend Dependencies

`@vitejs/plugin-react` npm development dependency needs to be updated to a version compatible with Vite 4. Change the version the `package.json` file:

.package.json
[source,json]
----
{
  "devDependencies": {
    "@vitejs/plugin-react": "~3.1.0"
  }
}
----
