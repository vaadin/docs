== Npm Dependencies

`@vitejs/plugin-react` dev dependency needs to be updated to a version comptatible with Vite 4. Change the version the `package.json` file:

.package.json
[source,json]
----
{
  "devDependencies": {
    "@vitejs/plugin-react": "3.1.0"
  }
}
----
