== Positioning

Cookie Consent can be positioned in the viewport in 4 non-stretched positions, or stretched across the top or bottom:

++++
<style>
  .align-right {
    text-align: right;
    display: block;
  }

  .align-center {
    text-align: center;
    display: block;
  }
</style>
++++

[cols="2"]
|===

|Top left
|[align-right]#Top right#

|Bottom left
|[align-right]#Bottom right#
|===

[cols="1"]
|===

|[align-center]#Top (stretch, default)#

|[align-center]#Bottom (stretch)#
|===

On smaller viewports, the component always takes up the entire width of the viewport.
