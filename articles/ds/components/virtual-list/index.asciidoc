= Virtual List

:components-path-prefix: ../

// tag::description[]
Virtual list displays a vertically aligned set of items inside a scrollable container.
Each item is rendered per need basis as the user scrolls the list, which makes the component especially suitable for displaying long item lists without sacrificing performance.
// end::description[]

To use the component, you need to assign it a set of data items and a renderer that's used for rendering each individual data item.
The height of an item is determined by its content and can change dynamically.
