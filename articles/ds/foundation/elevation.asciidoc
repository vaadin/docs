= Elevation

Elevation is used to indicate elements which are stacked on top of other elements in the UI.
Use these <<../customization/css-custom-properties#,CSS Custom Properties>> to create visually consistent elevation styles.

.Only available in Lumo
[NOTE,role=small]
This documentation applies only to the Lumo theme.

:import-name: lumo
:module-name: style
include::_shared.asciidoc[tag=importing]

== Box Shadow

Different levels of elevation are applied using box-shadow.

++++
