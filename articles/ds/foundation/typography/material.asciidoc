= Material Typography

See the Material Design documentation for https://material.io/design/typography/the-type-system.html[the type system], to learn about creating and applying a consistent typographic theme.

:import-name: material
:module-name: typography

== Font Family

++++
<style>
.custom-property-preview {
  font-family: var(--material-font-family);
}

.space.custom-property-preview::before {
  padding: calc(var(--value) * 0.5em);
}
</style>
++++

If available, the Roboto font is used. You should either include Roboto locally in your project, or import it from https://fonts.google.com/specimen/Roboto[Google Fonts].


[.property-listing.previews, cols="2,>3"]

|===
