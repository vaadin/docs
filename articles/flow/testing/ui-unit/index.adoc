= UI Unit Testing (Deprecated)

[NOTE]
====
UI Unit Testing has been renamed to *Browserless Testing* and is now free for all users starting with Vaadin 25.1. All new documentation is in the <<../browserless#,Browserless Testing>> section.
====

The `vaadin-testbench-unit` artifact and the `UIUnitTest` base classes continue to work but are deprecated. New tests should use the `browserless-test-junit6` artifact with the `BrowserlessTest` base class.

See <<../browserless/getting-started#,Getting Started with Browserless Testing>> to begin.
