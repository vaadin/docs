= Observability Kit

:commercial-feature: Observability Kit
:kit-trial: true
include::{articles}/flow/_commercial-banner.asciidoc[opts=optional]

Observability Kit provides integrations with industry-leading Application Performance Monitoring (APM) and observability tools to provide insight into both the front and backend of an application.

At its core, Observability Kit is a custom Java agent that runs together with a Vaadin Flow application to collect telemetry data, including traces, metrics, and logs.

.New in Vaadin 24.1.
[TIP]
You can now use Observability Kit <<{articles}/flow/tools/observability/configuration/#frontend-observability-configuration,to observe the frontend>> to observe the frontend of your application
