= Styling

include::../_styling-section-intros.adoc[tag=selectors]


Root element:: `vaadin-progress-bar`


=== States

Indeterminate:: `vaadin-progress-bar+++<wbr>+++**[indeterminate]**`


=== Parts

Track (bar):: `vaadin-progress-bar+++<wbr>+++**::part(bar)**`
Indicator (value):: `vaadin-progress-bar+++<wbr>+++**::part(value)**`


=== Style Variants

Success:: `vaadin-progress-bar+++<wbr>+++**[theme="success"]**`
Error:: `vaadin-progress-bar+++<wbr>+++**[theme="error"]**`
